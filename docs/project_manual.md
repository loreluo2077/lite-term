# Code with You: Lo-Fi Room 产品说明书

## 1. 产品概述

**产品名称**: Code with You: Lo-fi Room 
**产品定位**: 个人和AI Agent的协作终端平台

这是一款专为开发者和AI Agent协作设计的轻量级终端应用。cli-agent的工具现在已经非常成熟，而且各有特色,底层的大模型也是各有擅长的地方，但是我的痛点就在于cli-agent有非常多个，同时运行时，可能要打开多个cli并进行切换； 而且不管是启动方式，还是配置，现在还不统一化，而且对于在cli中操作，使用没有传统软件的体验那么好；而且现在的开发和之前已经不一样了，并不用要像之前的ide一样去写代码，现在已经慢慢从code变成了build，已经变成了另外一种沉浸式的体验，之前叫vibe coding，但我并不喜欢这个词，我更喜欢lofi这个词，所以我想追求符合我的开发体验，比如听着歌，刷着信息，玩会小游戏放松下脑子，然后验收agent们的工作，于是就有了Code with You: Lo-fi Room 


## 2. 已实现功能模块

### 2.1 工作空间 (Workspace)
- 支持多工作空间管理，便于组织不同的项目
- 每个工作空间可独立配置环境变量和启动脚本
- 工作空间状态持久化，支持应用重启后恢复

### 2.2 面板 (Panel)
- 可调节大小的分屏面板设计
- 支持多个终端会话同时显示
- 灵活的布局管理，支持垂直/水平分割

### 2.3 会话 (Session)
- 多终端会话管理
- 会话历史记录保存
- 会话标签页切换
- 会话状态恢复

### 2.4 AI Agent 协作
- 与AI Agent的无缝集成
- 在终端环境中直接调用AI能力
- 智能命令建议和自动补全

## 3. 核心特性

### 3.1 多窗口管理
- 采用现代化UI设计，支持多窗口布局
- 拖拽式窗口管理，灵活调整布局
- 键盘快捷键支持，提高操作效率

### 3.2 会话持久化
- 会话状态在应用重启后保持
- 自动保存会话历史和上下文
- 快速恢复工作流程

### 3.3 高效工作流
- 支持批处理命令执行
- 命令历史快速检索
- 丰富的自定义配置选项

## 6. 快速开始

### 6.1 安装
按照项目根目录中的安装指南进行安装配置。

### 6.2 基本使用
1. 启动应用后，创建一个新的工作空间
2. 在面板中打开新的终端会话
3. 开始使用命令行工具
4. 可选择与AI Agent协作完成复杂任务

## 7. 配置说明

### 7.1 工作空间配置
- 可设置工作空间名称、路径和默认启动脚本

### 7.2 面板配置
- 面板大小和位置调整
- 字体和主题设置
- 终端仿真器参数配置

### 7.3 会话配置
- 会话保存和恢复设置
- 历史命令数量限制
