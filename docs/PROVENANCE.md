## 使用规则

1. 仅当发生“直接复制后迁移”时才记录。
2. 纯参考思路或重写实现，记录在 `REUSE_MAP.md`，不在本文件登记。
3. PR 中新增直接复制代码但未更新本文件，视为不合规。
